## 核心概念速查

### 四大模型的对标场景

| 模型 | 最擅长 | 用途 | 风险 |
|------|--------|------|------|
| **DDM** | 高分红银行 | 长期价值投资 | 受增长率假设影响大 |
| **PB-ROE** | 价值洼地识别 | 快速筛选 | 只考虑静态ROE |
| **RIV** | 深度分析 | 组合配置 | 需要ROE预测 |
| **相对估值** | 择股决策 | 最终确认 | 需要同业数据 |

### 三个关键估值指标

1. **PB（市净率）** = 股价 / 每股净资产
   - A股银行平均：0.888x（当前处于历史低位）
   - 低于0.7x：严重低估
   - 0.7-1.2x：合理范围
   - 高于1.5x：严重高估

2. **ROE（净资产收益率）** = 净利润 / 净资产
   - A股银行平均：11.2%
   - 反映银行的赚钱能力
   - ROE越高，PB合理值越高

3. **股息率** = 每股股息 / 股价
   - A股银行平均：3.52%
   - 高于3%：有吸引力
   - 高于5%：非常有吸引力

## 投资决策流程

```
【第一步】获取行业水平
     ↓
平均PB = 0.888x
平均ROE = 11.2%
平均股息率 = 3.52%
     ↓
【第二步】PB-ROE模型快速筛选
     ↓
当前PB < 0.7x？ → YES → 进入低估池
            → NO → 继续分析
     ↓
【第三步】验证分红能力（DDM）
     ↓
合理PB > 当前PB？ → YES → 有吸引力
            → NO → 需谨慎
     ↓
【第四步】深度分析（RIV）
     ↓
内在价值高于当前价格？ → YES → 强烈建议
                   → NO → 观察中
     ↓
【第五步】相对估值确认
     ↓
在同类银行中是最便宜的？ → YES → 最终推荐
                    → NO → 比较分析
```

## 自定义配置

### 修改参数位置

文件：`config.py`

**最常修改的参数：**

```python
# 1. 股权成本（r）- 影响所有模型
VALUATION_PARAMS[\"equity_cost_base\"] = 0.10  # 默认10%
# 建议范围：8%-12%
# 当市场利率上升时，可增加到11%-12%

# 2. 永续增长率（g）- 影响DDM和PB-ROE
VALUATION_PARAMS[\"perpetual_growth_rate\"] = 0.04  # 默认4%
# 建议范围：2%-5%
# 保守估计可用3%

# 3. 如果要把严重低估的PB阈值改为0.8
THRESHOLDS[\"undervalued_pb\"] = 0.8  # 默认0.7
```

修改后重新运行即可生效。

## ⚠️ 重要提示

### 什么时候要远离银行股

🔴 风险信号：
1. **ROE持续下降** - 赚钱能力在衰退
2. **不良率快速上升** - 风险在积累（>2%需警惕）
3. **资本充足率下降** - 分红和扩张空间受限
4. **股息率异常高** - 可能有隐患
5. **行业政策变化** - 如利率大幅上升
